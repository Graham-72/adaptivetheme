
The views templates have additional classes to help global themeing.

The following 3 page level templates have thier extra classes added to the main wrapper.
views-view.tpl.php : skinr classes
views-view--page.tpl.php : skinr classes + .view-page-display
views-view--block.tpl.php : skinr classes + .view-block-display

views-view-list.tpl.php,
